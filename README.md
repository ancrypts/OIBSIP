# Oasis Infobyte Cyber Security Internship (OIBSIP)

## Overview

This repository contains all the tasks completed during my Cyber Security Internship at Oasis Infobyte.

The internship focuses on developing practical cyber security skills through hands-on activities such as network scanning, password security analysis, phishing awareness, and vulnerability assessment.

# Tasks

## Task 1
Basic Network Scanning

Learn how to discover active hosts, open ports, running services, and operating systems using Nmap.

---

## Task 2
Password Strength Analysis

Understand password complexity, entropy, password attacks, and methods to create strong passwords.

---

## Task 3
Phishing Awareness

Identify phishing emails, malicious websites, social engineering techniques, and prevention strategies.

---

## Task 4
Basic Vulnerability Assessment

Perform vulnerability scanning, analyze risks, identify weaknesses, and recommend mitigation measures.
