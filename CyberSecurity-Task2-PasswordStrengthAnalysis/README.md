# Project Overview

Password security is one of the most important aspects of cyber security. Weak passwords are one of the leading causes of unauthorized system access and data breaches.

This project demonstrates how password strength can be evaluated using an online password strength checker and security best practices.

---

# Objective

The objective of this project is to analyze different passwords and understand the factors that make passwords weak, moderate, or strong.

# Password Strength Factors

A password becomes stronger when it contains:

- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Longer length
- No dictionary words
- No personal information

# Findings

- Password length significantly improves security.
- Dictionary words reduce password strength.
- Special characters increase complexity.
- Passwords should be unique for every account.

---

# Recommendations

- Use passwords longer than 12 characters.
- Enable Multi-Factor Authentication (MFA).
- Avoid personal information.
- Never reuse passwords.
- Use a password manager.

---

# Conclusion

This project demonstrates how password strength affects account security. Strong passwords reduce the risk of brute-force attacks, dictionary attacks, and credential stuffing.
