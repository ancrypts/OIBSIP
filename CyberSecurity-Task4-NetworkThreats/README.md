Covered Threat Vector Focus Areas
This threat intelligence report reviews four critical vectors impacting modern enterprise networks:
1.	Distributed Denial of Service (DDoS): Resource depletion and service disruptions (SYN Floods, Amplification Attacks).
2.	Phishing & Social Engineering: Human credential harvesting and initial network access methods.
3.	Man-in-the-Middle (MitM): Network traffic sniffing, ARP poisoning, and cryptographic session hijacking.
4.	Ransomware Operations: Network infiltration, standard file encryption methods, and double-extortion strategies.
